## 🚀 Quick Start (3 Steps)

### Step 1: Import
```dart
import 'package:flocateapp/widgets/modern_widgets.dart';
```

### Step 2: Use
```dart
ModernGradientButton(
  label: "Click Me",
  onPressed: () {},
)
```

### Step 3: Customize
```dart
ModernGradientButton(
  label: "Save",
  icon: Icons.check,
  gradientColors: [Colors.purple.shade400, Colors.blue.shade400],
  onPressed: () {},
)
```
